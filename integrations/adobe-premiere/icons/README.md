# Panel Icons

This directory should contain the following icon files for the panel:

- `icon-normal.png` - 23x23px icon for light UI
- `icon-rollover.png` - 23x23px icon for light UI hover state  
- `icon-dark-normal.png` - 23x23px icon for dark UI
- `icon-dark-rollover.png` - 23x23px icon for dark UI hover state

All icons should be PNG format with transparency.

For now, you can use any 23x23px placeholder images or create simple icons with "MAMS" text.
